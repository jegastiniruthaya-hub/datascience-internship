# Multi Modal AI Assistant

## Project Description
This project is a Multi Modal AI Assistant that can understand both image and text inputs.

## Features
- Image Upload
- Image Caption Generation
- Question Answering
- Conversation Memory
- Evidence-Based Responses

## Technologies Used
- Python
- Streamlit
- Transformers
- BLIP Model
- Pillow

## How to Run

pip install -r requirements.txt

streamlit run app.py
